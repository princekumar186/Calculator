# 🧮 Calculator Web Application

A simple and responsive **calculator web application** built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations and provides an interactive interface similar to a real calculator.

The goal of this project is to demonstrate **JavaScript logic, DOM manipulation, and responsive UI design**.

---

## 🚀 Features

* Perform basic arithmetic operations
* Addition, Subtraction, Multiplication, Division
* Clear and delete functionality
* Responsive user interface
* Real-time input display
* Keyboard-like button interaction

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout
* **JavaScript** – Calculation logic and user interaction

---

## 📂 Project Structure

```
calculator/
│
├── index.html     # Calculator layout
├── style.css      # Styling
└── script.js      # Calculator logic
```
